# Vacancy Search Tool

## Основной функционал:

* Поиск вакансий по ключевому слову.
* Отображение информации о найденных вакансиях: название, ссылка, зарплата, описание, компания, город и опыт.
* Сохранение всех найденных вакансий и топ N вакансий по зарплате в JSON-файл.
* Фильтрация вакансий по ключевому слову в описании.

## Требования:

* Python 3.13 или выше
* Установленные зависимости с использованием Poetry.
